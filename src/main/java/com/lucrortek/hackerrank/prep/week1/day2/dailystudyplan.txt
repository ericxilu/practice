```java
// 1. Linear Search
public static int linearSearch(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) {
            return i;
        }
    }
    return -1;
}

// 2. Reverse Array
public static void reverseArray(int[] arr) {
    int left = 0;
    int right = arr.length - 1;

    while (left < right) {
        // Swap elements
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;

        left++;
        right--;
    }
}

// 3. Find Maximum
public static int findMax(int[] arr) {
    if (arr.length == 0) return Integer.MIN_VALUE;

    int max = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}
```

### **Common Mistakes to Watch For:**
- **Off-by-one errors**: Remember arrays are 0-indexed
- **ArrayIndexOutOfBoundsException**: Always check `i < arr.length`
- **Null pointer exceptions**: Check if array is null before accessing
- **ArrayList vs Array confusion**: `length` vs `size()`, `[]` vs `get()`
